Logic gate : It is an electronic circuit having one or more than one input and only one output. The relationship between the input and the output is based on a certain logic.
There are different kinds of logic gates. The project involved the following logic gates: AND Gate: The AND gate gives us a HIGH (1) output iff all the inputs to the AND gate are HIGH (1). Stuck at faults occur when a line is permanently stuck to Vdd or ground giving a faulty output. This line may be an input or output to any gate. Also this fault can be single or multiple stuck at faults. A fault model is an engineering model of something that could go wrong in the construction or operation of a piece of equipment. From the model, the designer or user can then predict the consequences of this particular fault. Basic fault models in digital circuits include the stuck-at fault model, the bridging fault model, the transistor faults, the open fault model, the delay fault model, etc. If a circuit has n signal lines, there are potentially 2n stuck-at faults defined on the circuit.
